React

What is React ?
- JavaScript based front-end development library
- mainly used for web application and mobile application development
- it was introduced in 2013 by Facebook
- it follows declarative approach, component based library that allows developer to build
reusable UI components
- it follows Virtual DOM approach to optimize performance by minimizing DOM updates
